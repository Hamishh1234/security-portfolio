# Activity A26 – System Bug

## Objective
To identify and demonstrate a simple system bug

## What I did
I was helping my girlfriend with a Python program that withdraws money from a balance.

While testing the program we tested different inputs to see how it behaved.

## Findings
I noticed that there was a bug in the code. When the user entered a negative number it would increase the size of the balance. Eg. entering -100 would add 100 to the balance instead of removing it. From this I realised we needed to not allow negative numbers (numbers bellow 0). I then fixed the code by added some new lines and taught her how it worked.
