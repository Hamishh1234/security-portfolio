# Activity A8 – Cryptography in IoT Devices

## Objective
To identify how cryptography is used in Internet of Things (IoT) devices

## What I did
I went through devices in my home that connect to the internet and tried finding any ways cryptography was used

## Findings
What i found was my home WiFi network which is used by a lot of devices in my house (TV, phones, laptops). The network uses encryption (such as WPA2) which protects data being sent between devices and the router. This ensures that the communication between devices is secure as if a hacker tries to intercept this data they will not be able to understand it. A lot of devices connect to the internet and that has to be a secure connection because of how common it is. If there were no encryption, lots of peoples data would be seen/stolen.

## Reflection
Cryptography plays a huge role in keeping our data safe. Our data is getting transmitted online all the time, which gives the risk of interception. Cryptography helps by mitigating a lot of the risk.
