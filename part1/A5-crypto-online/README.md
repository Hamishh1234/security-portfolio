# Activity A5 – Cryptography Online

## Objective
To identify how cryptography is used on a website

## What I did
I went onto the UWA website and looked at how it handles security when loading the page

## Findings
I noticed that the UWA website uses HTTPS which is shown in the url and also by the lock icon with "connection is secure" in the dropdown from the browser

HTTPS uses encryption to protect data being sent between the user and the website. This means that information like login details or personal data is not sent in plain text. Therefore if this data get intercepted by someone, they wont be able to understand what it says due to it being encrypted, which helps keep your data and information safe.

The browser shows that the connection is secure which confirms that encryption is being used.

## Evidence collected
Screenshot showing the HTTPS URL and the "connection is secure" dropdown on the UWA website

## Reflection
From this i noticed how common encryption is as its used on most new websites. It is constantly keeping peoples data and information safe without them noticing.
