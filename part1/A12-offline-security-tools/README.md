# Activity A12 – Offline Security Tools

## Objective
To identify different tools used for security in offline environments

## What I did
I looked at security tools that I found inspiration from around the house and online.

## Findings
A padlock is used to secure things like gates, lockers, or storage areas. They are a portable lock feature that has a U-shaped shackle that can go through a link or a chain to keep items secure. To unlock them you need a key.

A security camera is a camera that monitors an area and records activity. They record and transmit live video (and in some cases audio) feed to a monitor or system. They improve security as they can help you detect threats.

Motion sensor lights are lights that turn on when movement is detected. They are usually used outside houses so owners can be alerted when movement is detected within the set range. This improves security by helping detect a threat.

A safe is a secure lockable container and they are made to store valuable items and keep them protected. To unlcok them you usually need to know a combination or a code, which you enter into it.

A door chain is a second lock for your door. They add a second layer of protection by being a lock that is only unlockable from the inside. The chain connects to the wall next to where the door opens and when you want to lock it you connect it to the doorframe. This helps with security as it gives the owner the ability to look at who is outside without them being able to get in. This is because the door can slightly open when this is connected, but not enough to get inside.
