# Activity A1 – Campus Security

## Objective
To identify security concepts used on campus

## What I did
I walked around the campus and looked for any security features used 

## Findings
I found was a bike lock used on an electric scooter. This is used to prevent people from being able to take it unless they know the code/have the key to it.

Bollards (metal poles) are also placed in walkways. These are used to stop vehicles from entering certain areas and improve safety. But if they need their vehicles to be able to enter they can put them down so they can.

Security cameras were around campus and they are used to monitor activity around the campus. It gives them the ability to see recorded/live video.

I found a key lock on a door and that controls access into the building. You need a key to access.

I also found a card access reciever, where if you want to enter you have to swipe an authorized card.
