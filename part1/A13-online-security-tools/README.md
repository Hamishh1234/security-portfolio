# Activity A13 – Online Security Tools

## Objective
To identify and understand a range of online security tools

## What I did
I searched for different online security tools. I looked through different websites so i could learn what they do and how they improve secuirty.

## Findings
The online security tools i decided to use were:

- Bitwarden – A password manager that stores and manages passwords securely so then users don’t have to reuse passwords.
- NordVPN – A VPN service that encrypts internet traffic and helps protect user privacy online.
- Cloudflare – Provides security for websites including protection against DDoS attacks and it improves website performance.
- VirusTotal – Allows users to scan files and URLs to check for malware using multiple antivirus engines.
- Have I Been Pwned – Lets users check if their email or passwords have been involved in data breaches.

## Reflection
From this, I realised there are heaps of different tools that help with online security. Each tool focuses on different areas such as password management, privacy, or threat detection. Using a range of these tools (the more the better) helps increase your online security more (better then relying on just one).
