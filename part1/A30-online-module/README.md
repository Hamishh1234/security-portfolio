# Activity A30 – Online Cybersecurity Module

## Objective
To complete an online cybersecurity learning module

## What I did
I completed an online cybersecurity module that covered networking and security fundamentals. I found this one on the internet.

## Findings
The module explained how networks work and it included how devices communicate using IP addresses and protocols such as TCP and UDP. It also covered concepts like subnetting and how networks can be divided into smaller sections to improve security. It also went into MAC addresses and how ARP is used to map IP addresses to physical devices on a network. It also explained the difference between IPv4 and IPv6, and why IPv6 is needed due to the limited number of IPv4 addresses. It also covered basic cryptography and authentication methods.
