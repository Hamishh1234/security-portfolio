# Activity A7 – Cryptography in Modern Networks

## Objective
To identify how cryptography is used in modern networks

## What I did
I checked the security settings of my hotspot, which I was connected to and looked at how it protects the connection. To do this I tapped the WIFI logo in the top right on my mac while holding options.

## Findings
I found that my hotspot is using WPA2 Personal security. WPA2 is a type of encryption used in WiFi networks which protects data being sent between devices and the router. This helps stop other people from being able to read the data when intercepting it or accessing the hotspot without permission. This helps security as it helps stop people from being able to read sensitive information being sent through a network.
