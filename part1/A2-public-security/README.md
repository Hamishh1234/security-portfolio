# Activity A2 – Public Space Security

## Objective
To identify security concepts used in public spaces

## What I did
I was in a park so I looked around for security features I could find

## Findings
Padlocks were used to secure a door and an electrical box. This prevented unauthorized access into things/places.

A key lock was used on another metal box which stops unauthorized access into it. If you want to open it you would need a key.

Lights that turn on when it gets dark. These improve visability to help stop crime.

There was also a dead lock on a toilet that could only be locked and unlocked from the inside. This controls access so the person inside has control over it.
