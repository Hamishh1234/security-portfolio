# Activity A11 – Access Control Devices

## Objective
To identify different devices used to control access

## What I did
I searched online about different ways access is controlled in everyday situations. I picked examples that I have seen before so i understand them more.

## Findings
An example is a keypad door lock. This is where you enter a code to unlock a door. Some people decide to use these on their houses just because they are easier then remembering a key. People also use them in air-bnbs to make it possible to give people access without a physical key but stops anyone from just being able to use it. Unless you kmow the code, you cant get in.

Another example is a swipe card system. To gain access you need a card that is read by a reader. Using a magnetic strip or radio frequency the card stores user data. These are used in places like universities or gyms as they can store data such as who has scanned their card that day.

Fingerprint scanners work by scanning the unique ridges and valleys of a fingure. This converts them into a digital template and compares the data against stored records to verify identity. These control access as only people that are allowed in (their fingerprint digital template is on the database to be allowed in). A common place these are found are on phones and your fingerprint scanner can let you into the device, skipping having to type the passcode in.

A key fob is a small battery powered transmitter. These transmitters use RFID (Radio Frequency Identification) or radio signals to send data to a receiver. They look like a small remote with buttons. When a button is pressed it sends an encrypted code to the receiver and if it matches its code it will perform an action. These are used for car keys or garage remotes which let you unlock them without having to physically insert a key.

Intercom systems are used in apartments these are systems where someone inside has to approve access before the door opens. They are two way systems where communication devices (audio and remote door unclocking and in some cases video) are transmitted between "master stations" and "substations". They can use wires or wifi.
