# Activity A9 – Privacy Techniques Online

## Objective
To understand how privacy is protected when using the internet

## What I did
I looked at different ways websites and browsers try to protect user privacy.

## Findings

A common privacy technique is private browsing (incognito mode). This allows users to browse the internet without saving history, cookies, or login details on the device. Incognito mode is easy to do and it is effective. The most value I see out of incognito mode is that it doesnt save cookies. This is because cookies can be used to track users across sites. This effects user privacy. This is only one of many things cookies can do.

Another technique is the use of cookies and tracking controls. Websites use cookies to store information but users can block or manage them to improve privacy. Cookies are small bits of data that websites save in your browser and they remember things for you. They affect privacy by tracking users across sites and personalising ads based off what users look at. Cookies can stay for a long period of time. By rejecting cookies this can help increase user privacy.

HTTPS is a way which protects data being sent between the user and the website. HTTPS stands for HyperText Transfer Protocol Secure. HTTP is the non secure version and in this data is sent in plain text and can be intercepted. When HTTPS is used, data is encrypted so only the website can read it. This can add a extra layer of protection of sensitive information.

## Evidence collected
Information was gathered from online websites

## Reflection
This activity showed that there are simple and effective ways to improve privacy online but lots of people don’t use them. Small things like using private browsing or managing cookies can make a big difference.
