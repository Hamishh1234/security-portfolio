# Activity A14 – AI Security Solutions

## Objective
To identify security systems that use AI

## What I did
I looked at apps and websites I use regularly and thought about how they use AI to improve security

## Findings
Gmail’s spam filter automatically detects and filters suspicious emails into a spam folder. This is a use of AI because there isnt someone manually doing this. They have a built in AI that looks for patterns of spam emails.

YouTube content moderation is where videos or comments can be flagged or removed if they seem inappropriate, rude or they shouldnt be on there. This is done by an AI, when a user uploads a comment it runs it through AI to see if it is appropriate and if it doesnt pass the tests it wont be posted.

Messenger also has spam detection which filters out suspicious messages or requests from unknown users. It uses AI to see regular spam patterns and if a user matches these patterns there messages may be places into spam.

Google Chrome uses security warnings to alert users if a website may be unsafe or not secure. This is a use of AI because if the site has common features of an unsafe site the AI will pick up on this and tell the user.

CommBank also uses security features to detect unusual activity like suspicious transactions or logins. When it detects one of these it sends the user a email/message letting them know and if it the user doing it they might need to put a authentication code in for it to work.

All these use AI to detect threats without needing humans constantly looking over them.
