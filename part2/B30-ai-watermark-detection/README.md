# Activity B30 – AI Watermark Detection

## Objective
To generate an AI-created image, apply an imperceptible watermark, edit the image, and check whether the watermark survives.

## What I did
I generated an AI image of a mountain lake landscape.

I then added a very faint watermark to the image using the text “HH-PORTFOLIO-B30”. I made the watermark very low opacity so it was difficult to see normally.

After that, I edited the image by slightly cropping it, resizing it back, adjusting it slightly, and exporting it again.

## Findings
The watermark was not obvious when looking at the normal watermarked image.

To check if the watermark survived, I compared the edited watermarked image with an edited original control image. The difference between the two images was amplified so the watermark could be detected.

The detection image showed that the watermark was still present after the editing process.

This means the watermark survived the edit, even though it was not easily visible in the normal image.

## Evidence collected
I included the original AI image, watermarked image, edited versions, and the watermark detection image in the evidence folder.

## Reflection
This activity showed me how watermarking can be used to help identify AI-generated images. It also showed that a watermark can still be detected after an image has been edited, even if it is not obvious when viewing the image normally.
