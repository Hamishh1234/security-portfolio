# Activity B24 – Design Access Control

## Objective
To design and implement access control of my choice

## What I did
I created a Google Drive folder and changed the sharing settings so that access was restricted.

## Findings
I used Google Drive because it is a common tool where access control is used. When sharing files or folders, the owner can decide who is allowed to access them and what they are allowed to do.

For this task, I set the folder’s general access to “Restricted”. This means that the folder is not open to anyone with the link. Only people who are specifically added can access it.

This is an example of access control because it limits who can view or use the folder. If the folder was set to “Anyone with the link”, then someone could access it if they got the link. By keeping it restricted, the data is more private and secure.

## Evidence collected
I included a screenshot showing the Google Drive sharing settings set to “Restricted” instead of “Anyone with the link”.

## Reflection
This showed me how access control is used in normal applications like Google Drive. It made me realise that small settings can make a big difference, because sharing something the wrong way could expose private files to people who should not have access.
