# Website Security Recommendations

Website reviewed: Perth DJ Hub login page

I noticed that the login page did not show visible protections such as CAPTCHA or account lockout after multiple failed login attempts.

Recommended improvements:
- Add CAPTCHA after several failed login attempts
- Add temporary account lockout or login delays
- Send warning emails after multiple failed login attempts
- Add two-factor authentication for user accounts
- Monitor repeated login attempts from the same IP address

These changes would help reduce brute force login risks and improve account security.
